# Sign Language Recognition System

This project is a Sign Language Recognition System that uses Deep Learning and Computer Vision techniques to recognize hand gestures and translate them into corresponding alphabets or words in real time. The model was trained using a Convolutional Neural Network (CNN) to accurately classify sign language gestures from image data.

The system captures hand gestures through a webcam using OpenCV, preprocesses the images, and passes them to the trained CNN model for prediction. TensorFlow and Keras were used for building, training, and evaluating the model, while Python served as the primary development language.

## Features

* Real-time hand gesture detection
* Sign language alphabet recognition
* CNN-based image classification
* Webcam integration using OpenCV
* High accuracy gesture prediction
* User-friendly interface

## Technologies Used

* Python
* TensorFlow
* Keras
* OpenCV
* NumPy
* Matplotlib
* CNN (Convolutional Neural Network)

## Objective

The goal of this project is to facilitate communication by converting sign language gestures into readable text, helping bridge the gap between hearing-impaired individuals and the wider community.
